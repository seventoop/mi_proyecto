* Renderiza botón oficial de Google (GSI)

2. Flujo:

* El frontend envía ese token al backend vía POST

3. Backend:

* Verifica el JWT usando librería oficial de Google
* Valida:

  * firma
  * issuer
  * audiencia (client_id)
  * expiración

4. Datos:

* Extrae:

  * sub (google_id)
  * email
  * name
  * picture

5. Base de datos (Railway):

* Hace UPSERT en tabla users usando google_id como clave única

6. Sesión:

* Backend emite cookie de sesión segura:

  * HttpOnly
  * SameSite=Lax
  * Secure solo en producción

7. Logout:

* Endpoint para cerrar sesión
